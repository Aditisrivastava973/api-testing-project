# api-testing-project
REST API Testing using Postman (CRUD + Negative Testing)

This project demonstrates API testing using Postman.

## 🔧 Features
- CRUD Operations (GET, POST, PUT, DELETE)
- Response validation using scripts
- Negative testing (invalid data, empty body, wrong endpoint)

## 🛠 Tools Used
- Postman
- REST APIs (JSONPlaceholder)

## 📌 Test Coverage
- Status code validation
- Response time validation
- JSON structure validation

## 📁 Collection File
Download and import into Postman:
`api-testing-project.json`
